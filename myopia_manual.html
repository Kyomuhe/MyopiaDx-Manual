MyopiaAI: Pathological Myopia Diagnosis System

Welcome to MyopiaAI, an advanced artificial intelligence system designed to assist healthcare professionals in diagnosing pathological myopia. This user manual provides comprehensive guidance on using the system effectively, from logging in to generating diagnostic reports.

AI-Powered Analysis

Fast and accurate analysis of fundus and FANTAS images with advanced deep learning algorithms.

Patient Management

Comprehensive patient record management with secure data storage and easy access.

Detailed Reporting

Generate professional diagnostic reports with detailed findings and visualizations.

Statistical Analysis

Track and analyze diagnostic trends across demographics and severity levels.

Important: This system is intended for use by qualified medical professionals only. The AI provides decision support but does not replace clinical judgment.

Getting Started

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge - latest versions)
  • Stable internet connection
  • Minimum screen resolution of 1280x720

Accessing the System

1

Open your web browser and navigate to the MyopiaAI login page.

2

Enter your credentials:

  • Username
  • Password
3

Click "Login" to access the system.

First-Time Login

If this is your first time logging in:

  1. Use the temporary credentials provided by your administrator
  2. You will be prompted to change your password
  3. Create a new password following the security requirements:
    • Minimum 8 characters
    • At least one uppercase letter
    • At least one number
    • At least one special character

Password Recovery

If you forget your password:

  1. Click "Forgot Password" on the login screen
  2. Enter your registered email address
  3. Follow the instructions sent to your email to reset your password

Dashboard Overview

After logging in, you will be presented with the dashboard, which serves as the main control center of the system.

Key Dashboard Elements

Statistics Panel

This section displays key metrics and analytics:

  • Total patients diagnosed
  • Diagnoses by gender
  • Diagnoses by age groups
  • Diagnoses by severity levels
  • Diagnoses by religion (if applicable)
  • Recent diagnostic activity
Statistic Description Display Format
Gender Distribution Percentage breakdown of diagnoses by gender Pie chart
Age Distribution Distribution of patient ages in diagnostic database Bar chart
Severity Levels Breakdown of diagnoses by severity classification Donut chart
Religious Demographics Optional breakdown by religious affiliation Pie chart

Quick Access Menu

Located on the left side of the screen, this menu provides quick access to:

  • Patient Management
  • New Diagnosis
  • Reports Archive
  • System Settings
  • User Profile

Notification Center

Located in the top-right corner, this section displays:

  • System updates
  • New results ready for review
  • Administrative announcements

Search Functionality

A universal search bar allows you to quickly find:

  • Patients by name, ID, or other identifiers
  • Previous diagnoses
  • Specific reports

Customizing Your Dashboard

You can personalize your dashboard by:

  1. Clicking the "Customize" button in the top-right corner
  2. Selecting which statistics to display
  3. Rearranging panels using drag-and-drop
  4. Saving your preferences

Tip: Customize your dashboard to show the statistics most relevant to your clinical practice. Specialty clinics may want to focus on specific demographic information or severity metrics.

Patient Management

Adding a New Patient

1

Click "Add Patient" button on the dashboard or from the Quick Access Menu

2

Fill in the required patient information:

  • Full name
  • Date of birth
  • Gender
  • Contact information
  • Medical record number (if applicable)
  • Religion (if applicable)
  • Existing medical conditions
  • Current medications
  • Family history of eye conditions
3

Click "Save" to create the patient record

Searching for Existing Patients

  1. Use the search bar at the top of the dashboard
  2. Enter the patient's name, ID, or other identifier
  3. Click on the patient's name from the search results to access their profile

Editing Patient Information

  1. Navigate to the patient's profile
  2. Click "Edit" in the patient information section
  3. Update the necessary fields
  4. Click "Save Changes"

Patient Profile Overview

Each patient profile includes:

  • Personal information
  • Medical history
  • Previous diagnoses and results
  • Uploaded images
  • Generated reports
  • Recommended follow-ups

Important: Always verify patient identity before updating records or performing diagnostic procedures. Patient data is protected under applicable privacy laws and regulations.

Diagnostic Process

Initiating a New Diagnosis

  1. Navigate to the patient's profile
  2. Click "New Diagnosis" button
  3. Select the type of diagnosis to perform

Uploading Images

1

In the diagnosis screen, click "Upload Images"

2

Select the FANTAS image files from your computer

  • Accepted formats: JPEG, PNG, DICOM
  • Maximum file size: 20MB per image
3

Ensure the images are clear and properly oriented

4

Add any relevant notes about the images

Image Quality Requirements

For optimal diagnostic accuracy:

  • Images should be well-focused
  • Proper lighting should be used
  • Standard imaging protocols should be followed
  • Patient information should be correctly associated with the images

Tip: The AI system performs best with high-quality images. Poor image quality may result in lower confidence scores or diagnostic errors.

Initiating the AI Analysis

  1. Once images are uploaded, review them for quality and completeness
  2. Click "Start Analysis" button
  3. The system will display a progress indicator while the AI processes the images
  4. Wait for the analysis to complete (typically 1-3 minutes depending on image complexity)

Results and Reporting

Viewing Diagnostic Results

Once the AI analysis is complete:

  1. The system will display a summary of findings
  2. Detailed results will be presented with:
    • Diagnosis classification
    • Confidence scores
    • Highlighted areas of concern on the images
    • Comparison with reference images

Understanding the Diagnosis

The system categorizes pathological myopia findings into:

  • Severity levels (Mild, Moderate, Severe)
  • Specific conditions detected
  • Risk factors identified
  • Progression indicators
Severity Level Characteristics Typical Findings
Mild Early signs of pathological changes Minimal retinal thinning, early tessellation
Moderate Established pathological changes Posterior staphyloma, chorioretinal atrophy
Severe Advanced degenerative changes Lacquer cracks, Fuchs spot, CNV

Generating Reports

1

Review the AI-generated results

2

Click "Generate Report" button

3

Select the report format (Detailed Clinical Report, Summary Report, Patient Information Sheet)

4

Add any additional notes or observations

5

Click "Create PDF"

Downloading and Sharing Reports

  1. Once the PDF is generated, click "Download" to save it locally
  2. To share the report:
    • Click "Share" button
    • Enter recipient's email or select from system users
    • Add a brief message (optional)
    • Click "Send"

Report Archives

All generated reports are automatically archived and can be accessed:

  • From the patient's profile under "Reports" tab
  • From the dashboard's "Reports Archive" section
  • Via the search function using patient name or report date

Recommendations

AI-Generated Recommendations

Based on the diagnostic results, the system provides:

  • Treatment suggestions
  • Follow-up scheduling recommendations
  • Referral suggestions to specialists
  • Patient education materials

Customizing Recommendations

  1. Review the AI-generated recommendations
  2. Click "Edit Recommendations" to modify or add to the suggestions
  3. Save your changes
  4. These custom recommendations will be included in the final report

Note: The AI provides recommendations based on current medical guidelines and research, but clinical judgment should always take precedence.

Follow-up Scheduling

The system can help manage follow-up appointments:

  1. From the recommendations section, click "Schedule Follow-up"
  2. Select a recommended timeframe or enter a custom date
  3. Add notes about the purpose of the follow-up
  4. Save the follow-up to the patient's profile

Troubleshooting

Common Issues and Solutions

Login Problems

  • Ensure caps lock is off
  • Clear browser cache and cookies
  • Use the password reset feature if necessary

Image Upload Issues

  • Check that the file format is supported
  • Ensure the file size is under the limit
  • Try a different browser if problems persist

Analysis Errors

  • Verify image quality meets requirements
  • Check internet connection stability
  • Retry the analysis

Report Generation Problems

  • Ensure the analysis has completed successfully
  • Check browser permissions for downloading files
  • Try generating a different report format

Error Messages

The system provides specific error messages to help identify issues:

Error Code Description Recommended Action
Error 101 Login authentication issue Verify credentials and try again or reset password
Error 201 Image quality insufficient Retake image with better focus/lighting
Error 301 Analysis processing error Check connection and retry or contact support
Error 401 Report generation failure Retry with different format or contact support

For any error codes not listed, please contact technical support.
